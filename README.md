# Scripts for reconstructing the evolutionary history of genomic segments through ancestral WGD events

Ancestral genome reconstruction with examples for proto-vertebrate, gnathostome, and cyclostome.

## Building

An example bash script for building the executable can be found in `scripts/build.bash`. This may need to be modified based on your version of java, operating system, and computer architecture.

## Usage

Example usage for reconstructing the ancestral vertebrate and gnathostome genomes can be found in `scripts/vertebrate.bash` and `scripts/gnathostome.bash` respectively. Refer to Nakatani et. al., 2021 for additional details.
